# 01 Plan

## Problem Summary

현재는 활성 미니앱 구현 배치가 없다.

## Scope Compression

- 1-day MVP: 다음 요청이 들어오면 가장 작은 검증 가능한 미니앱 범위를 정한다.
- 3-day MVP: 다음 요청이 들어오면 사용자 경험을 더 풍부하게 만드는 확장 범위를 정한다.
- 1-week MVP: 다음 요청이 들어오면 메인 플랫폼 연결과 재사용 자산을 포함한 확장 범위를 정한다.

## Selected Scope For This Batch

현재 배치의 범위는 Oshiz Mini App 전용 하네스 구조를 준비하는 데까지다.
