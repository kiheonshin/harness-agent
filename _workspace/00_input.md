# 00 Input

## Current Request

현재는 활성 사용자 요청이 없다.

## Scope

- 다음 실제 작업 요청이 들어오면 이 파일에 요청과 범위를 정리한다.
- 기본 하네스에서는 특정 도메인 규칙 없이 일반적인 파일 기반 작업 하네스를 제공한다.

## Constraints

- 사용자 커뮤니케이션과 작업 문서는 한국어 중심으로 유지한다.
- 구현 전에는 계획과 검증 경로를 먼저 문서화한다.

## Acceptance Criteria

- 다음 작업 요청이 들어왔을 때 이 파일만 읽어도 요청, 범위, 제약, 성공 기준을 이해할 수 있어야 한다.
- 하네스는 특정 제품이나 프로젝트에 종속되지 않아야 한다.

## Open Questions

활성 작업이 시작되면 필요한 질문만 기록한다.
